¶

Classes

This section describes the base classes that the entity system provides for clients and other library modules. They provide a starting point for all major types of physics entities, from simple particles in space to complicated 3D meshes.

Entity

Class Entity describes a basic physics entity with position, velocity, and direction. This is the basic unit that all physics classes build from.
